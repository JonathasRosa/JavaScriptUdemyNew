Eventos - JS

O que são eventos?

Eventos são ações condicionadas a algum tipo de resposta feita pelo usuário, por exemplo: clicks, apertar alguma tecla ou até movimento do mouse. POdemos atrelar lógica a essas ações do usuário, por meio de handlers.

Como criar um evento

Devemos atrelar o evento a um elemento, por exemplo um botão, depois inserir um listener e um tipo de evento como argumento, ai o elemento responderá por este evento e obviamente, os outros da página não.

EX:

let bnt = document.querySelector("#bnt");
bnt.addEventListener("click", () => {
console.log("clicou");
});

Removendo eventos

Da mesma forma que podemos adicionar eventos, podemos remover quando achamos necessário. Para isso utilizamos o método removeEventListener, onde passamos o evento e a função que o evento está escutando.

EX:

let btn = document.querySelector("#btn");
function msg() {
console.log("clicou");
}
btn.addEventListener("click", msg);
setTimeout(function() {
btn.removeEventListener("click", msg);
}, 3000);

O Objeto do evento

Quando criamos eventos, temos a opção de utilizar um argumento, que é chamado de objeto do evento. ele contem prorpiedades que podem ser utilizadasa nosso favor, esse objeto é criado pelo JavaScript automaticamente.

EX:
function msg(e) {
console.log(e);
}
btn.addEventListener("click", msg);

Propagação

Quando não definimos um elemento muito bem(seletor brando) ou um elemento que está dentro de outro evento, pode acontecer a propagação que é um elemento ativando um evento de outro também, aí teremos uma duplicação. Para pararmos uma propagação e não acarretar uma série de problemas utilizamos o stopPropagation.

Ações default

Muito elementos/teclas já tem suas ações pré-definidas, como clicar num link que nos leva a outra página, mas podemos para este evento default e criar uma lógica diferente para elemento em questão.

EX:

let a = document.querySelector('a');
a.addEventListener('click', function(e) {
e.preventDefault();
console.log("clicou mas não mudou de link");
})

Eventos de tecla - Keyup e keydown

Sempre que uma tecla é pressionada, são gerados dois eventos, keyup/keydown, com isso podemos realizar ações nestes eventos também.

EX:

window.addEventLitener("keydown", function(e) {
if(e.key == "y"){
console.log("Apertou a tecla y");
}
});

Eventos de mouse

No mouse temos também eventos como mouseup e mousedown, semelhante aos da teclas, também temos o Dblclick que podemos ativar um evento com clickduplo.

EX:

btn = document.querySelector("button");
btn.addEventListener("dblclick", function() {
console.log("Ativou o double click");
});

Movimento do Mouse

Também podemos atvar eventos com a movimentação do mouse, o nome dele é mousemove através desse evento podemos detectar a posição do ponteiro do mouse na tela, muito CUIDADO pois ele pode sobrecarreagara a memória do computador, uma vez que o JavaScript utiliza o computador do cliente como recurso para ser executado. Porém é um evento muito interessante pos podemos atrelar eventos em posições específicas quando o mouse passar em determinado local.

EX:

window.addEventListener("mousemove", function(e) {
console.log(e.x);
console.log(e.y);
});

Eventos por Scroll

Podemos atrelar eventos ao scroll da tela também. pelo evento scroll, podemos criar um elemento assim que o o scroll atigir uma posição "X".

EX:

window.addEventListenert ("scroll", function(e) {
if(window.pageYOffset > 100) {
console.log("Chegou na posição");
}
});

Eventos por Foco

Quando focamos em um elemento ou saimos dele, podemos também atrelar um evento a esta ação. Focus quando um elemento recebe o foco e blur quando ele perde o foco.

EX:

let input = document.querySelector("input");
input.addEventListener("focus", function (){
console.log("foco no input");
});

input.addEventListener("blur", function (){
console.log("perdeu o foco");
});

Debouce

Um evento que dispara múltiplas vezes pode ser um problema para o computador do cliente, por isso podemos fazer um debounce que é um suavizador de evento, para não chamar a atençãodo usuário muitas vezes.

EX:

windo.addEventListener("mouse", function() {
clearTimeout(timeout);
timeout = setTimeout(() => console.log("Debounce!"), 500);
});
