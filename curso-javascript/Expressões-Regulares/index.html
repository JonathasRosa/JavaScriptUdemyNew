Expressões Regulares

O que são expressões regulares?

Uma forma de encontrar padrões em uma string, por exemplo, validar se só tem múmeros.

Em vez de criarmos funções comlexas, podemos encontrar dados em texto por meio das expressões regulares, também chamada de regex.

Criando uma expressão regular

As expressões regulares no JS são um tipo de objeto, podendo então instancia-lo de duas formas

  • New RegExp.
  • /.../ 'Dentro de barras' (sendo a mais utilizada).

EX:

let reg1 = new RegExp("test");
let reg2 = /teste/;

Verificando padrões com REGEX

Depois da definição do padrão por meio da REGEX, utilizamos o metodo test(), para verificar se o padrão é retornado,com isso vamos receber a resposta true ou false.

EX:
console.log(/testando/.test("ttttestandoooo")); console.log(/testando/.test("astestansd"));

Conjunto de caracteres

Podemos definir um conjunto de caracteres para encontrar por regex, basta por [] e com um traço podemos definir um intervalo também.

EX:

console.log(/[123]/.test("Existe 123 aqui?"))
console.log(/[0 - 9]/.test("O número 8 está presente aqui?"))

Caracteres especiais

  • \d - Qualquer digito de caractere;
  • \w - Um caractere alfanumérico("teste");
  • \s - Qualquer caractere de espaço em branco;
  • \D - Caractere que não são dígitos;
  • \W - Caractere não-alfanumérico
  • \S - caractere que não seja espaço em branco;
  • . - Qualquer caractere, menos nova linha.

Regex com caracteres especiais

Podemos utilizar os caracteres especiais de várias formas, veja.

EX:

let ano = /\d\d\d\d/;

console.log(ano.test("05"));
console.log(ano.test("2022"));
console.log(ano.test("opa"));

let palavraTresLetras = /\w\w\w/;

console.log(palavraTresLetras.test("dia"));
console.log(palavraTresLetras.test("ano"));
console.log(palavraTresLetras.test("oi"));
console.log(palavraTresLetras.test("teste"));

Operador not (^).

Podemos escrever um set que aceitam tudo, menosalguns caracteres com o not, lembrando que combinações serão aceitas, o negado é apenas se bate com o set.

EX:

let palavrasSemAeB = /[^123]/;

console.log(palavrasSemAeB.test("1112"));
console.log(palavrasSemAeB.test("14"));
console.log(palavrasSemAeB.test("1"));

Operador plus +

Quando um "+" está a´pos alguma expressão, este elemento pode se repetir mais de uma vez.

EX:

let muitosOuPoucosDigitos = /\d+/;

console.log(muitosOuPoucosDigitos.test("123"));
console.log(muitosOuPoucosDigitos.test("0123456789"));
console.log(muitosOuPoucosDigitos.test(" "));

Operador question "?"

Ele faz com que o dígito anterior seja opcional.

EX:

let opcional = /Prova\s?\d?/;

console.log(opcional.test("Prova"));
console.log(opcional.test("Prova 1"));
console.log(opcional.test("Prova 2"));
