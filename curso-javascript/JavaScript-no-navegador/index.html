A web

Protocolos

Um protocolo é uma forma de comunicação entre computadores através da rede.

Exemplos de protocolos:

  • HTTP, serve para nós solicitarmos arquivos, imagens e etc.(Hyper Text Transfer Protocol)
  • SMTP, protocolos para enviar e-mail.
  • TCP, protocolo de transferência.

O que é uma URL?

Cada arquivo que carrega no navegador é nomeado por uma URL que pode ser dividido em três partes

  • Protocolo - http // https;
  • jonathasrosa.github.io/Portifolio/projeto-cordel-digital - Servidor (DNS para um IP) onde o site é hospedado;
  • index.html - o arquivo que vai ser carregado no servidor e depois subir para a rede.

O arquivo HTML

HTML (Hyper Text Markup Language) é uma linguagem de marcação, responsável pelos textos e elementos que vemos na tela ao acessar uma página web.

As tags do HTML, servem para criar elementos como título, parágrafos, imagens, listas etc, dentro do arquivo.

Uma tag é definida por abertura e fechamento com a seguinte estrutura abre a tag com sinal de menor que "<" coloca o elemento dentro e fecha com o ">" sinal de maior que, isso abre uma tag, para fechar uma tag usa-se o sinal de menor que "<" depois uma barra vertical para direita "/" coloca o elemento dentro e fecha a tag com sinal de maior que ">" novamente.

Html - parte 2

Todo documento HTML tem sempre duas partes importantes, head e body.

O head serve para configurações do documento, como links de fora de estilo ou até de scripts de JavaScript, tags de configurações meta links entre outras.

O body para elementos, ou seja, a parte visual, "p, h1, div, ul, li,strong, dentre várias outras.

O HTML e o JavaScript

Podemos utilizar JavaScript no HTML por meio da tag "script", ou likar um arquivo .js por meio de script com o atibuto src(forma mais utilizada), algumas tags HTML aceitam atributos que executam JS, mas não é muito utilizado. Lembrando sempre que sempre que linkamos um arquivo externo, estamos chamando ele via HTTP.

O HTML e o DOM

Quando uma página web é reqisitada, ela recebe um texto que é transformado em uma estrutura HTML, as tags vão se aninhando uma às outras, formando uma estrutura em que elementos ficam dentro dos outros, essa estrutura tem uma representação exatamente igual para o JS chamado de DOM (Document Object Model) e por meio do Java Script podemos acessar cada um desses elementos do HTML através do DOM.

Entendendo o DOM

Podemos mudar completamente uma página HTML através do DOM, sendo possível alterar, elementos, atributos, CSS, além de alterar podemos adicionar e também remover elementosm além de ser possível criar eventos na página, como animações.

A árvore do DOM

O docmento HTML seria uma árvore completa, a raiz da árvore é o documento em si (tendo somente uma raiz), cada elemento da árvore chamamos de node (nó - head, title, body, tags), conteúdos como texto são chamados de leaf nodes(as folhas)

Acessando elementos/tags

Podemos acessar todos os elementos a partir de 'document' a partir dele vamos entrando nos childNodes (nós-filhos) e depois acessando as propriedades que nos interessasm.

EX:

  • console.log(docment.body);
  • console.log(docment.body.childNodes);
  • console.log(docment.body.childNodes[1]);
  • console.log(docment.body.childNodes[1].childNodes[1]);
  • console.log(docment.body.childNodes[1].childNodes[1].innerText);

Encontrando elementos

Podemos encontrar elementos de uma forma mais fácil do que se movendo pelo DOM, como no HTML temos tags, class e ids, o JavaScript nos possibilita utilizar essas caracteristicas para encontrar o que desejamos

Existe alguns metodos para isso:

  • console.log(document.getElementByTagName('h1'));
  • getElementById
  • getElementByClassName
  • querySelector

Encontrando elementod por ID

Utilizamos o seguinte método para encontrar elementos por ID.

EX:

  • console.log(document.getElementById('titulo-principal);

Encontrando elementos por class.

Utilizamos o segunte método para encontrar elementos por class.

EX:

  • console.log(document.getElementsByClassName('teste'));

Encontrando elementos por Query

Um método que se assemelha a grande funcionalidade jQuery, nos permitindo encontrar elementos pelo seus seletores de CSS, que facilita muito a nossa vida.

  • console.log(document.querySelector('#algun-id'));
  • console.log(document.querySelector ('.teste'));
  • console.log(document.querySelector('h1'));

Alterando o HTML

Podemos mudar quase tudo atravé do DOM, adicionar, mover, clonar elementos, utilizando métodos como: insertBefore, appendChild, replaceChild.

Alterando a DOM com insertBefore

Insere um nó antes do nó de referência do método, precisamos de um elemento para ser adicionado, que terá outro elemento antes dele e o elemento pai destes dois.

EX:

let span = document.createElement('span');
let el = document.querySelector('h1');
let pai = el.parentNode;

pai.insertBefore(span, el);
