O que é um programa/Software

É um conjunto de declarações(statements).

Statements são um conjunto de expressões.

Expressões é todo o fragmento de código que produz um valores.

Software tem um 'objetivo'.

Statements 'guiam' o software para seu objetivo.

Expressões são os valores que os statements esperam para guiar o software.

Nós já vimos statements e expressões de forma simples

Linha de comando ou statement com uma expressão dentro: console.log(1>2) dois valores

Expression ou expressão com um unico valor dentro: console.log('batata')
