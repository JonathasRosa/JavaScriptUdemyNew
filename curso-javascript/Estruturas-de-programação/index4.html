Funcionamento do ambiente

Como funciona o ambiente ?

Quando qulaquer programa é iniciado, um ambiente é criado e este ambiente não pe vazio ele contem funcões nativas da linguagem.

Estas FUNÇÕES são blocos de códigos que nos retornam um valor ou ma ação.

O ambiente de JavaScript, poderiamos entender como o navegador

A estrutura de uma função

Antes de mais nada : funções sçao chamadas durante o programa.

Você também pode ouvir: executar, chamar, invocar uma função.

E nós chamamos uma função utilizando o nome dela + abrindo e fechando parênteses;

  • nomeDaFnção();

Tambêm podemos inserir parâmetros, em algumas funções são brigatórios.

Basicamente são valores que podem mudar o resultado da função;

  • nomeDaFunção(parametro1, parametro2);

Função built-in:

1º - prompt( )

É uma das funções que é iniciada junto com o JavaScript e recebe um input do usuário do sistema podendo guardar este valor;

EX: let idade = prompt('Qual sua idade?');
console.log(idade)

Esta função é pouco utilizada.

2º - alert( )

Essa função exibe uma mensagem na tela do usuário

EX: alert('Veja esta mensagem')

Esta função também é pouco utilizada.

Math.x( )

Função utilizada pa expressões/cálculos matemáticos.

Ex: let maiorNumero = Math.max(1,5,9,7,8,3);
console.log(maiorNumero);

Esta função é bastante utilizada.

console.log

Exibe uma mensagem na tela, que é o argumento que passamos.

EX: console.log('exibindo esta msgem')

É uma função muito utilizada.
