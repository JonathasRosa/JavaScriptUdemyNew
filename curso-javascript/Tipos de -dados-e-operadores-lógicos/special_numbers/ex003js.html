Special Numbers

São palavras que são consideradas numbers, mas não funcionam como números

Infinity, quando temos uma operação aritimética errada gerando um valor infinito.

-Infinity,quando temos uma operação aritimética errada gerando um valor -infinito.

NaN (Not A Number), quando somamos alguma coisa que é um "número" com algo que não é número, ou trata alguma coisa que não é número como número, consequentemente o JavaScript diz que aquilo não é número executando o NaN.

  • console.log(Infinity);
  • console.log(-Infinity);
  • console.log(NaN);
  • console.log(typeof Infinity);
  • console.log(typeof -Infinity);
  • console.log(typeof NaN);
