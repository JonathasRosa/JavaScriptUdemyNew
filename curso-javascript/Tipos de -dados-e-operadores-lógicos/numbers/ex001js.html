Numbers: aritimética

As operações feitas com os numbers em JS, funcionam da mesma forma que na matemática.

console.log(typeof 5);
console.log(typeof 21.7);
console.log(typeof -123);
console.log(2 + 2);
console.log(2 * 2 - 3);
console.log(8 / 4);
console.log(5 + (2 * 4));

Os resultados das operações aritiméticas em JS produz um novo Number.

Numbers: aritimética operadores

  • + = soma;
  • - = subtração;
  • / = divisão;
  • * = multiplicação;
  • % = resto;
