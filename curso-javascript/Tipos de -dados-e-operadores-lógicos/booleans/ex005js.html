Booleans

Serve para guardar um valor de uma comparação, por exemplo.

Os unicos valores possíveis são:

  • True (verdadeiro);
  • False (Falso);
  • console.log(5>2);
  • console.log(3>10).

Tipos de comparações

  • Maior que: >
  • Menor que: <
  • Maior ou = : >=
  • Menor ou = : <=
  • Igual: ==
  • Diferente: !=
  • Idêntico: ===

Operadores lógicos

Por meio de uma comparação resultam em Boolean.

  • && - and => para ser true os dois lados da comparação precisam ser true;
  • || - or => para ser true basta um dos lados ser true;
  • ! - not => inverter os valores (True vira false - false vira true);
A B A and B A or B NOT A
False False False False True
False True False True True
True False False True False
True True True True False

Boolean: Operador Ternário

Faz um comparativo em apenas uma linha de código

  • console.log(true ? 1:2);
  • console.log(false ? 'false' : 'verdadeiro')
  • Obs: Não é muito utilizado e pode deixar o código complicado de ler, não vale a pena ser usado.
