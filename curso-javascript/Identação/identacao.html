Continue

Dependo da nossa lógica, podemos pular um determinado "bloco" de execução, dando assim continuidade ao restante do execução.

let x = 0

while(x < 1) {
x = x + 1;
if(x % 2 == 0) {
continue;
}
console.log(x);
x = x + 1;
}
