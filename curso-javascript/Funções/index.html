Funções

Funções são estruturas de códigos menores que são reaproveitadas durante a execução/construção de um programa.

Principal objetivo é poupar repetições de código e podem ser consideradas 'subprogramas'.

Uma função tem uma estrutura um pouco mais complexa, deve ser declarada com a palavra reservada FUNCTION e essa função deve ter um nome, contendo argumentos/parâmetros, definitos entre "()" o corpo da função é definido entre chaves "{}", geralmente uma função retorna um valor, sendo pssível também declarar funções em variáveis.

EX:

function escreverNoConsole() {
console.log("Escrever no console!")
}
escreverNoConsole();
const textoNoConsole = function() {
console.log("Texto no console!")
}
textoNoConsole();
const textoPorParametro = function(texto) {
console.log(texto);
}
textoPorParametro("Testando por parâmetro!")

Três tipos de utilização da função

  • Normal sem argumentos.
  • Com argumentos.
  • Anonima atrelada a uma variável.
