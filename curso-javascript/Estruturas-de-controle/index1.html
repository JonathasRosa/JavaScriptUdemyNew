Estruturas de controle

Estruturas de controle significa que o código é executado de cima para baixo, com as estruturas podemos modelar o fluxo do programa, ou seja, dependendo dos valores de statements e expressions, ele tomará caminhos diferentes.

IF

O programa vai exeutar um bloco de código, SE algo acontecer.

Onde algo é a condição imposta por um statement.

onde o resultado é um booleans(true or false).

let idade =19;
if(idade > 18) {
console.log('Pode entrar');
}

ELSE

O bloco de código else é sempre utilizado com o bloco if caso a instrução do bloco if não seja atendida, ou seja, se não executar o bloco if, execute o bloco else.

let nome = 'jonathas' if(nome == 'João') { console.log(`Seu nome é ${nome}.`) } else { console.log('O seu nome não é João') }

Else If

Pode também chamar condições como else if, ou seja antes de executar um else, ou até mesmo sem ele podemos verificar mais de uma condição

let a = 5; let b = 3; if (a + b == 3) { console.log('O resultado é 3'); }else if (a == 4) { console.log('O valor de a é 4'); }else if (b == 3) { console.log('O valor de b é 3'); } else { console.lg('Nenhuma das condições acima!'); }
