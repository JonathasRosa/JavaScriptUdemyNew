Estruturas de controle

A idéia é repetir uma ação até atingir uma determinada condição, ao invés de repetirmos o mesmo código várias vezes, criamos um statement que fará uma checagem em cada LOOP.

A estrutura de repetição também é chamada de loop.

While

While fará uma ação, até que a condição seja atinginda.

Precisamos realmente "definir um fim" para o loop, para não termos o prolema de loop infinito, causando assim o travamento da página.

let x = 10

while(x > 0) {
console.log(x);
x = x -1;
}

Do while

Semelhante ao While, porém a estrutura muda um pouco e quase não é utilizado.

let y = 0;

do {
console.log(y);
y = y + 1;
} while( y < 5);

For

Semelhante ao While, porém amplamente utilizado sendo a estrutura de repetição mais fácil de compreender, sendo assim dê preferencia pelo For.

for (let numero = 2; numero < 100; numero = numero * 2) {
console.log("O numero é: " + numero)
}
